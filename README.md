# Youtube MP3 extractor
Youtube MP3 audio extractor from a CSV list

### Prerequisites:

Before running the script, you need to make sure that you have installed below components on your system:

1. __Python 3__: (Usually pre-installed on macOS).

2. __yt-dlp__: The download tool.

	- Installation (_Homebrew on macOS_): __brew install yt-dlp__

3. __FFmpeg__: Required by yt-dlp for MP3 extraction and conversion.

	- Installation (_Homebrew on macOS_): __brew install ffmpeg__

### Launching:

`
python3 audio_extractor.py ~/Desktop/lista_brani.csv ~/Desktop/MP3
`
